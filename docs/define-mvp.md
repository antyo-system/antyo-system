# MVP — Pomodoro Timer

## Deskripsi
Pomodoro timer sederhana untuk meningkatkan fokus dan produktivitas.

## Fitur MVP
- [x] Timer countdown 25 menit (fokus) dan 5 menit (istirahat)
- [x] Tombol Start / Pause / Reset / Skip
- [x] Switch otomatis antara mode fokus dan istirahat
- [x] Progress ring visual
- [x] Counter pomodoro yang sudah selesai
- [x] Dark mode support

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
